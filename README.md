# 🌟 English Text Analyzer

**AI-Powered English Text Analysis Tool for Educational Purposes**

🌐 **Live Demo**: [https://reasonofmoon.github.io/Text-Analysis/](https://reasonofmoon.github.io/Text-Analysis/)

영어 텍스트의 복잡도와 가독성을 분석하여 교육적 인사이트를 제공하는 종합 분석 도구입니다.

## 🚀 빠른 시작

1. **웹 앱 방문**: [https://reasonofmoon.github.io/Text-Analysis/](https://reasonofmoon.github.io/Text-Analysis/)
2. **API 키 발급**: [Google AI Studio](https://makersuite.google.com/app/apikey)에서 무료 Gemini API 키 발급
3. **텍스트 분석**: API 키 입력 후 영어 텍스트를 붙여넣고 "Analyze Text" 클릭
4. **결과 내보내기**: HTML, PDF, JSON 형식으로 보고서 다운로드

## 🎯 주요 기능

### 📊 **5가지 포괄적 분석 모듈**

#### 🔤 어휘 분석 (Vocabulary Analysis)
- **CEFR 레벨 분류**: A1~C2 자동 분류 및 수준별 권장사항
- **학술 어휘 식별**: Academic Word List 기반 학술 어휘 추출
- **연어 표현**: 자연스러운 단어 조합 및 관용 표현
- **어휘 다양성**: 텍스트의 어휘 풍부도 측정

#### 📝 문법 분석 (Grammar Analysis)
- **문장 유형**: 단문, 복문, 중문, 복합문 분류
- **시제 패턴**: 동사 시제 사용 분포 및 패턴
- **능동/수동태**: 문장 구조 복잡도 분석
- **구문 복잡도**: 고급 문법 구조 식별

#### 🏗️ 구조 분석 (Structure Analysis)
- **텍스트 조직**: 단락 구성 및 주제문 식별
- **응집성**: 논리적 연결성 및 일관성 분석
- **전환 표현**: 담화 표지어 및 연결어 추출
- **텍스트 흐름**: 아이디어 전개 패턴 분석

#### 💡 내용 분석 (Content Analysis)
- **주요 아이디어**: 핵심 주제 및 중심 논제 추출
- **논증 구조**: 주장, 근거, 반박 구조 분석
- **증거 유형**: 예시, 통계, 전문가 의견 분류
- **수사적 요소**: 설득 기법 및 어조 분석

#### 📊 복잡도 분석 (Complexity Analysis)
- **가독성 점수**: Flesch-Kincaid Grade Level, Reading Ease
- **CEFR 레벨 추정**: 유럽 언어 학습 프레임워크 기준
- **어휘 복잡도**: 고빈도/저빈도 어휘 비율
- **구문 복잡도**: 문장 길이 및 절 밀도

## 🚀 배포 및 설치

### 🌐 즉시 사용 (권장)
웹 브라우저에서 바로 사용: [https://reasonofmoon.github.io/Text-Analysis/](https://reasonofmoon.github.io/Text-Analysis/)

### 🏠 로컬 개발 환경
```bash
# 레포지토리 클론
git clone https://github.com/Reasonofmoon/Text-Analysis.git
cd Text-Analysis

# 웹 서버 실행
python -m http.server 8000 -d docs
# 브라우저에서 http://localhost:8000 접속

# 또는 Python 패키지 설치
pip install -r requirements.txt
pip install -e .
```

### 🐳 Docker 실행
```bash
# Docker 이미지 빌드 및 실행
docker build -t english-text-analyzer .
docker run -p 8000:8000 english-text-analyzer
```

## 🌟 사용 예시

### 📝 기본 사용법
1. **텍스트 입력**: 분석할 영어 텍스트를 입력창에 붙여넣기
2. **모듈 선택**: 필요한 분석 유형 선택 (기본: 모든 모듈)
3. **분석 실행**: "Analyze Text" 버튼 클릭
4. **결과 확인**: 각 모듈별 상세 분석 결과 검토
5. **보고서 내보내기**: 원하는 형식으로 결과 다운로드

### 🎓 교육 현장 활용
```
교사: 새 단원의 읽기 자료 선정
→ 후보 텍스트들을 분석하여 학생 수준에 맞는 자료 선택
→ 어려운 어휘 미리 파악하여 사전 학습 계획
→ 문법 포인트 식별하여 수업 목표 설정
```

### 🔬 연구 활용
```
연구자: 교과서 텍스트 복잡도 분석
→ 학년별 교과서 텍스트 수집
→ 일괄 분석을 통한 난이도 진행 패턴 파악
→ 국제 기준과 비교하여 적절성 평가
```
